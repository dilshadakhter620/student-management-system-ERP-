# Student Management System ERP

A simple **Student Management System (ERP)** built to manage student records efficiently.  
This project allows adding, updating, viewing, and managing student data in a structured way.

---

## Features

- Add new students with details like name, roll number, class, and contact info
- Update existing student information
- Delete student records
- View all student details in a structured format
- Simple and user-friendly interface

---

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript (React optional)
- **Backend:** Python/Django (if backend included)
- **Database:** SQLite/MySQL (as per your setup)
- **Version Control:** Git & GitHub
